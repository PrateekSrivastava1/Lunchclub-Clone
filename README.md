## Getting Started

### Prerequisites

- **NPM**

```
npm install npm@latest -g
```

### Installation

1. Clone the repo

```
https://github.com/PrateekSrivastava1/Prateek_Srivastava_webDevelopmentAssignment_frontend_july2021.git
```

2. Install NPM packages

```
npm install
```
